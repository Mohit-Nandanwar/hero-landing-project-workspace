🚀 Digital Workspace Platform - Landing Page
A modern AI-powered workspace landing page built with HTML + Tailwind CSS.
This project demonstrates responsive design, sticky navigation, hero section layout, floating info cards, and a fixed chat widget.

✨ Features
Sticky navigation bar with logo and links
Hero section with headline, subtext, and CTA buttons
Responsive design for mobile and laptop screens
Floating information cards (tasks, project status, team activity)
Dashboard preview image centered in hero section
Fixed chat button linking to WhatsApp group
